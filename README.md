# Spring Blog Application

Spring Blog is a backend RESTful application built using **Spring Boot** that demonstrates real-world backend concepts such as authentication, authorization, security, JWT, role management, and API versioning.

This project is designed for **learning, practice, and interview preparation**.

- **Authentication**
  - Secure user login and registration

- **Authorization**
  - Role-based access control (RBAC)

- **Roles**
  - Fine-grained permission management for APIs

- **JWT Security**
  - Token-based authentication
  - Stateless API security

- **OncePerRequestFilter**
  - Custom token validation on every request
  - Acts as an interceptor for JWT verification

- **Spring Security**
  - Secured endpoints
  - Custom authentication and authorization logic

- **Exception Handling**
  - Centralized exception handling using `@ControllerAdvice`
  - Meaningful error responses

- **JPA & Database**
  - `JpaRepository` for data access
  - Optimized database interactions

- **JPA Auditing**
  - Automatically manages:
    - `createdBy`
    - `updatedBy`
    - `createdDate`
    - `updatedDate`

- **API Versioning**
  - Supports multiple API versions for backward compatibility

- **Image Uploading**
  - Upload and manage images/files

- **ObjectMapper**
  - Custom JSON serialization and deserialization
